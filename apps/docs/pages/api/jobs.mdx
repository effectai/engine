# Jobs

A job is a batch of tasks posted to the Effect AI worker network. Workers complete tasks and submit answers; you poll for results.

Credits are **debited upfront** at job creation (`reward × taskCount`). If a job fails to create after the debit, credits are automatically refunded.

## Job types

| Type | How it works | Best for |
|---|---|---|
| `csv` | One task per CSV row, each row seen by one worker | Labelling, classification, transcription |
| `constant` | Same data sent to up to `maxTasks` different workers | Surveys, repeated annotation, consensus tasks |

## Estimate cost

`POST /api/v1/jobs/estimate`

Validates a job and returns the cost without charging your account.

### CSV estimate

:::code-group

```bash [curl]
curl -s -X POST "https://api.effect.ai/api/v1/jobs/estimate" \
  -H "Authorization: Bearer eff_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "csv",
    "templateId": "TEMPLATE_ID",
    "reward": "0.5",
    "csv": "product,question\nWidget A,Is this good?\nGadget B,Is this good?"
  }'
```

```js [JavaScript]
const res = await fetch('https://api.effect.ai/api/v1/jobs/estimate', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer eff_live_YOUR_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    type: 'csv',
    templateId: 'TEMPLATE_ID',
    reward: '0.5',
    csv: 'product,question\nWidget A,Is this good?\nGadget B,Is this good?'
  })
})
const estimate = await res.json()
```

## Create a job

`POST /api/v1/jobs`

Creates a job, debits your credits, and immediately begins posting tasks to workers.

### CSV job

:::code-group

```bash [curl]
curl -s -X POST "https://api.effect.ai/api/v1/jobs" \
  -H "Authorization: Bearer eff_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "csv",
    "name": "Sentiment Labelling",
    "templateId": "TEMPLATE_ID",
    "reward": "0.5",
    "csv": "product,question\nWidget A,Is this good?\nGadget B,Is this good?",
    "uniqueWorker": false,
    "timeLimitSeconds": 600
  }'
```

```js [JavaScript]
const res = await fetch('https://api.effect.ai/api/v1/jobs', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer eff_live_YOUR_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    type: 'csv',
    name: 'Sentiment Labelling',
    templateId: 'TEMPLATE_ID',
    reward: '0.5',
    csv: 'product,question\nWidget A,Is this good?\nGadget B,Is this good?',
    uniqueWorker: false,
    timeLimitSeconds: 600
  })
})
const job = await res.json()
console.log(job.id) // use this to poll for status and results
```

```python [Python]
import requests

res = requests.post(
    'https://api.effect.ai/api/v1/jobs',
    headers={'Authorization': 'Bearer eff_live_YOUR_KEY'},
    json={
        'type': 'csv',
        'name': 'Sentiment Labelling',
        'templateId': 'TEMPLATE_ID',
        'reward': '0.5',
        'csv': 'product,question\nWidget A,Is this good?\nGadget B,Is this good?',
        'uniqueWorker': False,
        'timeLimitSeconds': 600
    }
)
job = res.json()
job_id = job['id']  # use this to poll for status and results
```

:::

**CSV job fields**

| Field | Required | Description |
|---|---|---|
| `type` | Yes | `"csv"` |
| `name` | No | Display name for the job (defaults to `"Untitled job"`) |
| `templateId` | Yes | ID of the template to use |
| `reward` | Yes | EFFECT reward per completed task (0.001–100) |
| `csv` | Yes | CSV string — first row is header, max 10,000 rows |
| `uniqueWorker` | No | `true` prevents the same worker completing multiple tasks |
| `timeLimitSeconds` | No | Seconds a worker has to complete a task before re-queue (default 600) |
| `capability` | No | Filter to workers with a specific capability tag |

:::note
The CSV must contain a column for every field the template uses; missing columns are rejected with `400 invalid_request`. The column name `dataffect/reward` and any column starting with `__effect` are reserved and rejected.
:::

:::

**Constant job fields**

| Field | Required | Description |
|---|---|---|
| `type` | Yes | `"constant"` |
| `name` | No | Display name (defaults to `"Untitled job"`) |
| `templateId` | Yes | Template ID |
| `reward` | Yes | EFFECT reward per completed task |
| `maxTasks` | Yes | How many workers should answer (cost = reward × maxTasks, max 10,000) |
| `data` | Yes | Object with a key for each template field |
| `uniqueWorker` | No | `true` ensures each worker answers only once (recommended for surveys) |
| `timeLimitSeconds` | No | Seconds per task before re-queue (default 600) |
| `capability` | No | Capability filter |

```json

  "reward": "0.5",
  "taskCount": 2,
  "uniqueWorker": false,
  "createdAt": 1750000100000,
  "tasks": {
    "queued": 2,
    "active": 0,
    "completed": 0,
    "failed": 0
  },
  "credits": {
    "currency": "EFFECT",
    "reserved": "1",
    "consumed": "0",
    "refunded": "0",
    "remaining": "1"
  }
}
```

:::note
`status` is either `"active"` or `"cancelled"`. There is no `"completed"` status: a job is done when `tasks.queued + tasks.active === 0`.

```js [JavaScript]
  'https://api.effect.ai/api/v1/jobs/JOB_ID/results?format=csv',
  { headers: { Authorization: 'Bearer eff_live_YOUR_KEY' } }

```

```python [Python]
import requests

res = requests.get(
    'https://api.effect.ai/api/v1/jobs/JOB_ID/results',
    headers={'Authorization': 'Bearer eff_live_YOUR_KEY'},
    params={'format': 'csv'}
)

    f.write(res.text)
```

:::

CSV columns: `taskId`, `submittedAt`, `worker`, `input`, `result`

:::note
Worker submissions are untrusted, so cells starting with `=`, `+`, `-`, or `@` are prefixed with a single quote (`'`) in the CSV download. This prevents spreadsheet formula injection when the file is opened in Excel or Google Sheets.
:::

## Cancel a job

`POST /api/v1/jobs/:id/cancel`

Stops the job immediately: no further tasks are posted to workers, and tasks currently in progress are not collected. Credits are refunded for **every task without a completed submission** at the moment of cancellation, both queued and in-progress. You only pay for tasks that were already completed.

Idempotent — cancelling an already-cancelled job returns `200`.

:::code-group

```bash [curl]
curl -s -X POST "https://api.effect.ai/api/v1/jobs/JOB_ID/cancel" \
  -H "Authorization: Bearer eff_live_YOUR_KEY"
```

```js [JavaScript]
const res = await fetch('https://api.effect.ai/api/v1/jobs/JOB_ID/cancel', {
  method: 'POST',
  headers: { Authorization: 'Bearer eff_live_YOUR_KEY' }
})
const job = await res.json()
console.log(job.credits.refunded) // EFFECT refunded
```

```python [Python]
import requests

res = requests.post(
    'https://api.effect.ai/api/v1/jobs/JOB_ID/cancel',
    headers={'Authorization': 'Bearer eff_live_YOUR_KEY'}
)
job = res.json()
print(job['credits']['refunded'])  # EFFECT refunded
```

:::

**Response `200`**

```json
{
  "id": "01JQKX3Z8VFNWRB2TGHM4Y5P6B",
  "name": "Sentiment Labelling",
  "type": "csv",
```
